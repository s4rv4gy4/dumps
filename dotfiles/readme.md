documentation:
- [.config](.config) : this folder contains `fish`, `nvim`, and `wezterm` configuration files
  - [.config/fish](.config/fish) : contains all fish related configs
  - [.config/nvim](./config/nvim) : contains all neovim related configs
  - [.config/wezterm](./config/wezterm) : contains all wezterm related configs
- [.ssh](.ssh) : this folder contains ssh related to `git`, and `servers` related configuration files
  - [.config/.ssh](./config/ssh) : contains ssh related config
  - `Note:` make sure you create folders for your git_profiles. like: `shit` for personal, and `work` for work
  - `Warning`: after creating git_profiles folders, all add there important files (see below)
    - Please add all files: `allowed_signers` - for signer, `ssh` - for ssh_priv_key, `ssh.pub` - for ssh_pub_key, `sign` - for sign_priv_key, `sign.pub` - for sign_pub_key 
- [git](./git) : this folder contains only `.gitconfig` related files
  - [git/shit/.gitconfig](git/shit/.gitconfig) : contains personal profile configuration
  - [git/work/.gitconfig](git/work/.gitconfig) : contains work profile configuration
- [.gitconfig](.gitconfig) : this file is only for `global git` configuration
